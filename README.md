## 特点

- 小鹤双拼
- 8105 个标准常用字
- 无生僻字豆腐块
- luna_pinyin 词库 + 按需外挂
- Shift 临时英文模式，比如 v ⇧ 2ex.com ⏎
- 暗色主题
- 默认关闭：emoji

## 安装

- 备份原配置
- [最新鼠须管下载](https://dl.bintray.com/rime/squirrel/)
- 替换 /Library/Input Methods/Squirrel/Contents/SharedSupport/opencc/TSCharacter.ocd 目录下的同名文件
- 复制其他文件到 ~/Library/Rime/
- 鼠须管菜单，点击部署与同步
